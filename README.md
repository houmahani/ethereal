# **Three.js Starter Boilerplate**

A minimal boilerplate for Three.js projects, powered by **Vite.js** for fast development. Includes a rotating cube example and modern best practices.

## **Features**

- ⚡ **Vite.js** for lightning-fast builds and live reloading.
- 🎨 **Tweakpane** for real-time parameter tweaking.
- 📐 Modular architecture with singletons for easy extensibility.
- 🌟 Example: A rotating cube with a configurable UI.

---

## **Getting Started**

### 1. Clone the Repository

```bash
git clone https://github.com/houmahani/threejs-boilerplate.git
cd my-folder
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Development Server

```bash
npm run dev
```

### 4. Build for Production

```bash
npm run build
```
